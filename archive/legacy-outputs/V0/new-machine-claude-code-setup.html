זה לא Skill, וזה בכוונה

שלב מחשב חדש הוא pre-skill. קודם מתקינים Claude Code או Codex, מחברים GitHub ומוודאים שהטרמינל עובד. רק אחר כך יש טעם להשתמש ב-Skill. ה-Skill הראשון שנבנה בהמשך יהיה לפתיחת פרויקט חדש מ-PRD/SPEC עד ריפו עובד.

כלל Bun

Bun מומלץ לפרויקטים חדשים. בפרויקט קיים, מתחילים לפי ה-lockfile הקיים: `package-lock.json` שייך ל-npm, ו-`pnpm-lock.yaml` שייך ל-pnpm. לא מחליפים package manager באמצע בלי בדיקה.
