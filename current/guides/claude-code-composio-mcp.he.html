חיבור כלים חיצוניים ל-Claude Code עם Composio

MCP הוא הפרוטוקול שמאפשר ל-Claude Code "לגעת" בכלים שלכם. Composio הוא הגשר שמקל את החיבור לכל הכלים האלה בקליק אחד.

סרטון הסבר סרטון מלא על חיבור כלים חיצוניים ל-Claude Code: youtu.be/_jOR8y4QYhc

רגע — מה זה MCP?

הרקע

Model Context Protocol — תקן פתוח לחיבור סוכנים לכלים

MCP (Model Context Protocol) הוא תקן פתוח שפרסמה Anthropic ב-2024. הוא מגדיר "שפה משותפת" שבאמצעותה סוכן AI יכול לדבר עם כלים חיצוניים.

בלי MCP: Claude יודע לכתוב אימייל, אבל לא יכול לשלוח אותו.

עם MCP: Claude מחובר ל-Gmail ויכול לשלוח, לחפש, לארגן — הכול מממשק אחד.

כל חיבור כזה נקרא MCP Server. Composio מספקת עשרות MCP Servers מוכנים מראש.

הזרימה: מ-Composio לתוך Claude Code

1

מחברים כלים ב-Composio

נכנסים ל-Composio, בוחרים Gmail / Notion / Slack ועוד, ומאשרים גישה. הכול עם OAuth — בלי לשמור סיסמאות.

2

Composio יוצר MCP Server

Composio מייצרת URL ייחודי שמתנהג כ-MCP Server. הוא כולל את כל הכלים שחיברתם בשלב 1.

3

מוסיפים ל-Claude Code

מדביקים את ה-URL בקובץ הגדרות Claude Code. מעכשיו Claude Code "רואה" את כל הכלים.

הגדרה שלב אחר שלב

לפני שמתחילים:
  • Claude Code מותקן על המחשב (ראו מדריך ההתקנה)
  • חשבון Composio חינמי: composio.dev — צרו לפני שממשיכים
שלב 1

צרו חשבון Composio

היכנסו ל-composio.dev וצרו חשבון חינמי.

בחרו את הכלים שאתם רוצים לחבר — לדוגמה Gmail, Google Calendar, Notion, Slack, Linear, GitHub ועוד.

לכל כלי, Composio יבקש אישור גישה דרך OAuth — תהליך מוכר של "כניסה עם Google".

שלב 2

השיגו MCP URL

בממשק Composio, נווטו אל MCP Servers ולחצו Create MCP Server.

בחרו אילו כלים לכלול. Composio יחזיר URL בצורה:

https://mcp.composio.dev/your-unique-token

שמרו את ה-URL הזה — תזדקקו לו בשלב הבא.

שלב 3

הוסיפו את ה-MCP ל-Claude Code

כיצד מוצאים את קובץ ההגדרות:

קובץ ה-MCP של Claude Code נמצא בתיקיית הבית שלכם:

  • Windows: C:\Users\<שם המשתמש שלכם>\.claude.json
    (לדוגמה: C:\Users\Gamestation\.claude.json)
  • Mac: ~/.claude.json

כיצד מגיעים לשם ב-Windows:

  1. פתחו File Explorer
  2. לחצו על שורת הנתיב למעלה ומחקו את התוכן
  3. הקלידו: %USERPROFILE% ולחצו Enter
  4. תראו תיקייה עם שם המשתמש שלכם (Desktop, Documents, וכד')
  5. חפשו קובץ בשם .claude.json

אם הקובץ לא קיים:

  • פתחו Notepad
  • שמרו קובץ חדש בנתיב %USERPROFILE%\.claude.json
    (חשוב: שנו את "Save as type" ל-"All Files" כדי שהסיומת תהיה .json ולא .json.txt)

אם הקובץ קיים — פתחו אותו ב-Notepad ומצאו את הסוגר הראשי {. הכניסו את בלוק ה-mcpServers בתוך הסוגריים הקיימים.

{
  "mcpServers": {
    "composio": {
      "command": "npx",
      "args": ["-y", "@composio-dev/mcp@latest"],
      "env": {
        "COMPOSIO_API_KEY": "המפתח-שלכם-מ-Composio"
      }
    }
  }
}
הסבר על JSON JSON הוא פורמט קובץ טקסט מובנה. חשוב: אל תוסיפו פסיק אחרי הסוגר האחרון } — זה יגרום לשגיאה.

לאחר השמירה: סגרו את Claude Code לגמרי (כולל מהטריי בשורת המשימות) ופתחו אותו מחדש.

בדיקה

ודאו שהחיבור עובד

בתוך Claude Code, כתבו:

list the available tools you have access to

Claude Code אמור להציג את כל הכלים שחיברתם דרך Composio.

לאחר מכן בדקו עם בקשה קצרה:

what are my last 3 emails from today?

אם Claude Code מחזיר מידע מ-Gmail — החיבור עובד.

מה עם אבטחה? Composio לא שומרת סיסמאות — היא עובדת עם OAuth tokens בדיוק כמו שהכלים עצמם מבקשים. אפשר לבטל גישה בכל עת מהגדרות Composio, ולחלופין מהגדרות הכלי עצמו (לדוגמה, מ-Google Account Security). Claude Code רק מדבר עם ה-MCP Server — הוא לא שומר אישורים בכלל.
